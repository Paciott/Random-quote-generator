# Progetto - Random Quote Generator

## Descrizione

Lo scopo del progetto Ã¨ quello di creare un generatore di citazioni tramite API request. Il progetto deve mostrare una prima citazione all'apertura della pagina, deve inoltre aggiornarsi e mostrare una citazione diversa ogni volta che viene cliccato il pulsante "Get random quote".

Al click del pulsante dovranno cambiare sia il colore dello sfondo della pagina che il colore del pulsante stesso.

Nel caso in cui dovesse verificarsi un errore nell'API request, la pagina deve tamponare l'errore e mostrare un'apposita schermata.
